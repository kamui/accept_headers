# AcceptHeaders

**AcceptHeaders** is a ruby library that aids in the parsing and sorting of
http accept headers.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'accept_headers'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install accept_headers
